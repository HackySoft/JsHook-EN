# runtime

用于获取当前hook运行时相关基础信息

## runtime.appInfo

`返回值`: ApplicationInfo

## runtime.packageName

`返回值`: string

## runtime.processName

`返回值`: string

## runtime.classLoader

`返回值`: ClassLoader

## runtime.isKernel

是否在内核模式中

`返回值`: boolean

## runtime.coreVersionCode

获取当前jshook的版本号

`返回值`: int

## runtime.modVersion

获取当前firamod的版本

`返回值`: string

## runtime.modVersionCode

获取当前firamod的版本号

`返回值`: int
